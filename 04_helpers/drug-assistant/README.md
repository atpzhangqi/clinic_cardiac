# Drug Assistant

这个模块适合整理你最常用、最容易混淆、最需要快速确认的药物信息。

## 推荐维护维度

- 通用名 / 商品名
- 药物分类
- 常见适应证
- 常用起始剂量与调整要点
- 肾功能 / 肝功能注意事项
- 关键禁忌证
- 需要监测的指标
- 重要相互作用
- 数据来源与更新时间

## 数据位置

- [../../05_data/drugs/drug_catalog.csv](../../05_data/drugs/drug_catalog.csv)

## 使用建议

- 先记“常用场景”，再慢慢补全字段
- 不确定的剂量内容，宁可空着，也不要凭记忆硬填
- 更新时只改数据源，说明页尽量少重复
