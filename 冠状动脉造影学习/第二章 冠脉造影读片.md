## 1. 右冠状动脉造影读片

### 1.1 右冠状动脉的走行

#### 核心走行

- 右冠状动脉（RCA）起始于主动脉窦（Valsalva 窦）中的右冠状动脉窦。
- RCA 开口位置通常比左冠状动脉（LCA）开口略低。
- RCA 从右心房与肺动脉之间穿过，随后进入右房室沟。
- 近端可发出：
  - 圆锥支（CB, conus branch）
  - 窦房结支（SN, sinus node branch）
- RCA 主干沿房室沟走行，到达锐缘支（AM, acute marginal branch）附近。
- 走行过程中通常发出 2-3 支右室支（RV, right ventricular branch）。
- RV 中通过锐缘部、走行于右心室下壁的血管称为 AM。
- 经过锐缘部后，RCA 主干从心脏膈面向左心室方向走行。
- 其中：
  - 走行于后室间沟的部分为右后降支（4PD）
  - 发出房室结支（4AV）后的部分为右后侧支（4PL）

#### 缩写速查

| 缩写 | 含义 | 备注 |
|---|---|---|
| RCA | 右冠状动脉 | Right coronary artery |
| LCA | 左冠状动脉 | Left coronary artery |
| CB | 圆锥支 | 常见于 RCA 近端 |
| SN | 窦房结支 | 可由 RCA 发出 |
| RV | 右室支 | 通常 2-3 支 |
| AM | 锐缘支 | 走行于右心室下壁 |
| 4PD | 右后降支 | Posterior descending branch |
| 4PL | 右后侧支 | Posterolateral branch |
| 4AV | 房室结支 | AV nodal branch |

### 1.2 右冠状动脉的评价

#### 标准投照体位

- RCA 摄影的标准投照体位主要有 3 种：
  - LAO 60°
  - RAO 30°
  - 头位与左前斜位的结合：头位 30° + LAO 30°
- 通常通过上述 3 个体位即可观察 RCA 的整体图像。
- LAO 60° 特别适合评价 RCA 整体情况。
- 如果很难将 C 形臂调整到 LAO 60°，调整至 LAO 45° 也可用于评价。

## 右冠不同节段的评价要点

### 近段（Seg.1）

- 评价包含 RCA 开口在内的近段病变时，建议使用 LAO 60°。
- 如果使用 RAO 30°，该部位血管走行容易与投照轴线重叠，导致评价困难。
- 评价近段末梢、尤其是通过房室沟附近的部分时，需要分别使用：
  - LAO 60°
  - RAO 30°

### 中段（Seg.2）

- RCA 在中段继续沿右房室沟向远端走行。
- 与近段末梢类似，中段评价应结合两个体位：
  - RAO 30°
  - LAO 60°
- 双体位观察有助于避免走行重叠造成的狭窄遗漏。

- 图示：RCA 近段（Seg.1）和中段（Seg.2）有狭窄。
- 重点：RAO 30° 有助于观察中段局部狭窄，但仍需与 LAO 60° 互相印证。

### 远段（Seg.3）

- RCA 主干远段走行方向与 RAO 30° 投照体位接近同一轴线。
- 因此不宜主要采用 RAO 30° 评价远段病变。
- 远段病变宜选择 LAO 60° 投照体位。

#### 图 2-1-4：RCA 远段造影（LAO 60°）

- 图示：RCA 远段（Seg.3）有狭窄。
- 重点：LAO 60° 可更好展开远段血管，避免 RAO 30° 下因轴向重叠而低估病变。

### 后侧支与后降支：Seg.4（PL, PD）

- 为评价 RCA 主干以及 4PL、4PD 分叉处形态，可使用 LAO + 头位的投照体位。
- 对后降支末梢病变，可通过 RAO 造影图像进行评价。
- 不宜采用 LAO 造影图像评价后降支末梢病变，因为：
  - LAO 投照体位与该血管走行方向几乎在同一轴线上。
  - 后降支末梢病变在 LAO 图像上可能呈现缩短。
  - 评价时可能出现遗漏。

#### 图 2-1-5：RCA 后侧支和后降支造影（LAO-头位）

- 图示：可清楚看到后降支（4PD）有连续狭窄病变。
- 重点：LAO-头位适合观察 RCA 主干、4PL/4PD 分叉及其形态。

## 本节读片提示

- RCA 读片要先建立“开口 - 右房室沟 - 锐缘部 - 膈面 - 4PD/4PL”的连续走行框架。
- 同一狭窄在不同投照体位下表现可能不同，评价时应避免只依赖单一角度。
- 近段和中段常需 LAO 60° 与 RAO 30° 对照。
- 远段更适合 LAO 60°。
- 后降支末梢病变在 LAO 下可能被缩短或漏诊，需注意 RAO 角度补充观察。
