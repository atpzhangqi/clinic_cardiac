# Drug Data

`drug_catalog.csv` 是药物助手的数据底表，适合后续继续扩展。

建议优先补充你最常用的 20 到 30 个药物，而不是一开始追求全量。

字段说明：

- `generic_name`: 通用名
- `drug_class`: 药物分类
- `common_indications`: 常见适应证
- `starting_dose`: 常用起始剂量
- `dose_adjustment`: 调整要点
- `renal_notes`: 肾功能相关提示
- `hepatic_notes`: 肝功能相关提示
- `contraindications`: 关键禁忌证
- `monitoring`: 需要监测的内容
- `interactions`: 重要相互作用
- `source`: 信息来源
- `updated_at`: 最近更新日期
