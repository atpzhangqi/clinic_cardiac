# [[瓣膜病]]介入手术方法与适应症

## Clinical Question

- 这篇笔记聚焦成人[[心内科]]日常最常见的[[瓣膜介入]]方式，以及“什么时候该想到介入、什么时候仍以外科为主”。
- 截至 `2026-04-05`，本文主要参考 [[2025 ESC/EACTS 瓣膜病指南]]；美国路径用 [[2020 ACC/AHA 瓣膜病指南]] 作为补充对照。

## Why It Matters

- [[主动脉瓣狭窄]]、[[二尖瓣反流]]、[[二尖瓣狭窄]]、[[三尖瓣反流]] 已进入“[[外科]] + [[经导管介入]]”并行时代。
- 临床真正需要先判断的，往往不是“能不能做”，而是：
  - 是不是已经到了“该干预”的时间点
  - 更适合 [[外科手术]] 还是 [[经导管治疗]]
  - 解剖是否适合，风险是否可接受
  - 是否应该尽早转诊到 [[Heart Team]] / [[Heart Valve Centre]]

## Core Framework

遇到疑似需要[[瓣膜介入]]的患者，先按这个顺序想：

1. 先确认是不是“[[重度瓣膜病]]”，以及病变机制是[[原发性]]还是[[继发性]]。
2. 再看有没有[[症状]]、[[左室功能下降]]、[[右室功能下降]]、[[心衰住院]]、[[肺动脉高压]]等触发干预的信号。
3. 再分流到“[[外科]]优先”还是“[[经导管]]优先”。
4. 最后才看具体是哪一种术式：[[TAVI]]、[[M-TEER]]、[[PMC]]、[[T-TEER]]、[[Valve-in-Valve]]、[[PVL closure]] 等。

## 一页速记

| 方法 | 主要对象 | 日常最该记住的适应症 |
| --- | --- | --- |
| [[TAVI]] | 重度[[主动脉瓣狭窄]] | 已达干预指征，且年龄、寿命预期、解剖和手术风险支持经导管路径 |
| [[Balloon Aortic Valvotomy]] | 重度[[主动脉瓣狭窄]] | 主要作桥接，不是常规终末方案 |
| [[PMC]] / [[PMBC]] | 风湿性重度[[二尖瓣狭窄]] | 有症状，且形态合适、无[[左房血栓]]、无中重度[[二尖瓣反流]] |
| [[M-TEER]] | 重度[[二尖瓣反流]] | 原发性 MR 高危不宜外科；继发性 MR 在[[GDMT]] / [[CRT]]后仍症状明显 |
| [[TMVI]] | 复杂[[二尖瓣病]] | 主要用于精选高危患者，如严重 [[MAC]] 背景下重度瓣膜功能障碍 |
| [[T-TEER]] / [[经导管三尖瓣治疗]] | 重度[[三尖瓣反流]] | 高风险、症状明显、药物优化后仍重度 TR，且无重度 RV 功能衰竭或前毛细血管性 PH |
| [[Valve-in-Valve]] | 生物瓣衰败 | 中高外科风险且解剖合适时优先考虑 |
| [[PVL closure]] | [[瓣周漏]] | 有临床显著反流和/或溶血、且解剖适合封堵 |

## 1. [[主动脉瓣]]介入

### [[TAVI]] / [[TAVR]]

最常用场景是[[重度主动脉瓣狭窄]]已达到干预阈值。

常见触发点：

- 有[[症状]]的重度 AS
- 无症状但已出现[[LVEF]]下降
- 无症状但属于高风险进展人群，如极重度 AS、跨瓣流速进展快、[[BNP]]明显升高、运动试验提示异常

[[2025 ESC/EACTS]] 对术式选择的实用记忆：

- `>=70 岁`、三叶式主动脉瓣狭窄且解剖合适：[[TAVI]] 推荐
- `<70 岁` 且低外科风险：[[SAVR]] 推荐
- 其余患者：由 [[Heart Team]] 结合寿命管理、解剖、通路和并存疾病决定 [[TAVI]] 还是 [[SAVR]]

[[2020 ACC/AHA]] 的常用记忆：

- `<65 岁` 或寿命预期 `>20 年`：更偏向 [[SAVR]]
- `>80 岁` 或寿命预期 `<10 年`，且可行经股路径：更偏向 [[TF-TAVI]]
- `65-80 岁`：[[SAVR]] 与 [[TF-TAVI]] 共同决策

临床提示：

- [[TAVI]] 不是“老年 AS 一律做介入”，仍要看[[解剖适合度]]、[[冠脉开口高度]]、[[瓣环]]、[[通路条件]]、是否[[双叶式主动脉瓣]]
- [[双叶式主动脉瓣狭窄]] 可以做 [[TAVI]]，但更偏向“精选病例 + 有经验中心”
- [[Balloon Aortic Valvotomy]] 主要是桥接到 [[TAVI]] / [[SAVR]]，或为紧急高风险非心外科手术争取时间，不是常规长期方案

## 2. [[二尖瓣狭窄]]介入

### [[PMC]] / [[PMBC]] / [[Percutaneous Mitral Commissurotomy]]

这是风湿性[[二尖瓣狭窄]]最经典、最应熟记的介入方式。

适合优先想到 [[PMC]] 的场景：

- 临床重度[[风湿性二尖瓣狭窄]]，通常 [[MVA]] `<=1.5 cm2`
- 有[[症状]]
- 无不利于 [[PMC]] 的解剖或临床特征

做 [[PMC]] 前要主动排除：

- [[左房血栓]]
- 超过轻度的[[二尖瓣反流]]
- 严重或双侧交界钙化
- 没有交界融合
- 合并需要外科处理的其他瓣膜病或需搭桥的[[冠心病]]

无症状但也应提前考虑 [[PMC]] 的情况：

- 血栓栓塞风险高
- 有新发或阵发性[[房颤]]
- 静息[[肺动脉压]]升高
- 妊娠或准备妊娠
- 近期要做重大非心外科手术

临床提示：

- [[退行性二尖瓣狭窄]] / 严重 [[MAC]] 不是 [[PMC]] 擅长的对象
- 对严重 [[MAC]] 合并重度瓣膜功能障碍的高危患者，[[TMVI]] 只属于精选病例策略，必须在复杂[[二尖瓣]]经验中心讨论

## 3. [[二尖瓣反流]]介入

### [[M-TEER]] for [[原发性二尖瓣反流]]

对[[原发性 MR]]，主线仍然是“能做 durable repair 的外科修复优先”。

但以下情况要想到 [[M-TEER]]：

- 有症状的重度[[原发性二尖瓣反流]]
- 解剖适合夹合
- 外科风险高，或不适合外科

一句话记忆：

- [[原发性 MR]]：[[外科修复]]优先，[[M-TEER]] 主要给高危外科患者

### [[M-TEER]] for [[继发性二尖瓣反流]]

这是日常最容易用到的另一个场景。

考虑 [[M-TEER]] 前必须先做足：

- [[GDMT]] 已优化
- 该做的 [[CRT]] 已完成或已评估不适合
- 重新评估后仍为重度[[继发性 MR]]

最实用的筛选点：

- 仍有[[症状]]
- [[LVEF]] 大致在 `20%-50%`
- [[LVESD]] `<=70 mm`
- [[PASP]] / [[SPAP]] 不要太高
- 没有重度 [[RV dysfunction]]
- 没有已经进入晚期、难以从减轻 MR 中获益的[[终末期心衰]]

一句话记忆：

- [[继发性 MR]]：先药物和器械优化，之后仍症状明显、参数合适，再想 [[M-TEER]]

## 4. [[三尖瓣]]介入

### [[T-TEER]] / [[经导管三尖瓣治疗]]

这是近几年最值得补课的方向之一。

[[2025 ESC/EACTS]] 的日常实用结论：

- 对高风险、症状性、重度[[三尖瓣反流]]患者
- 在[[最佳药物治疗]]后仍明显受限
- 且没有重度[[右室功能障碍]]或前毛细血管性[[肺动脉高压]]

此时应考虑 [[经导管三尖瓣治疗]]，目标主要是改善[[生活质量]]和[[右室重构]]。

一句话记忆：

- [[重度 TR]] 已不再只有“利尿 + 等外科”两条路；如果外科风险高，要主动想到 [[T-TEER]]

## 5. [[生物瓣衰败]]与再介入

### [[Valve-in-Valve]]

这类场景在既往做过[[生物瓣置换]]的患者里会越来越常见。

应优先想到 [[Valve-in-Valve]] 的情况：

- 已出现有症状的显著生物瓣功能障碍
- 不是[[瓣膜血栓]]导致
- 外科再手术风险中到高
- 解剖与原瓣型号适合经导管再植入

[[2025 ESC/EACTS]] 实用要点：

- [[主动脉位 Valve-in-Valve]]：中高外科风险、解剖合适时应考虑
- [[二尖瓣位 Valve-in-Valve]] / [[三尖瓣位 Valve-in-Valve]]：中高外科风险且解剖合适时应考虑

## 6. [[瓣周漏]]介入

### [[PVL closure]]

什么时候想到：

- 有临床显著[[反流]]
- 出现[[溶血]]
- 或已经造成[[心衰症状]]

一句话记忆：

- 对解剖合适的[[瓣周漏]]，经导管封堵已经是成熟路径，不必默认都去开胸再修

## 临床最实用的床旁判断

### 什么时候应尽快转到 [[Heart Team]]

- 重度[[主动脉瓣狭窄]]且已经有症状
- 重度[[原发性 MR]]，但年龄大、合并症多、外科风险高
- [[继发性 MR]] 在[[GDMT]]后仍反复[[心衰住院]]
- 重度[[三尖瓣反流]]长期水肿、腹胀、肝淤血，但又不像适合单纯药物拖延
- 既往[[生物瓣]]开始衰败，怀疑要做 [[Valve-in-Valve]]

### 什么时候不要急着把病人往介入上推

- 还没确认是否真的是[[重度病变]]
- [[继发性 MR]] 还没做足[[药物优化]] / [[CRT]]
- 有明显不利解剖条件，成功率和获益都可疑
- 已存在重度终末期[[左室]] / [[右室]]衰竭，减轻瓣膜病变未必还能带来实质获益

## Common Pitfalls

- 把[[经导管治疗]]理解成“比外科轻，所以谁都能做”
- 没分清[[原发性 MR]] 和[[继发性 MR]]，导致治疗路径错位
- 看到[[三尖瓣反流]]只会加利尿，而没有想到近年[[经导管三尖瓣治疗]]适应症已经前移
- 把 [[PMC]] 用到不合适的[[退行性二尖瓣狭窄]]或重度钙化病变上
- 忽略[[寿命管理]]：年轻患者第一次瓣膜干预，不能只看当下创伤小不小，还要看下一次、下下次怎么处理
