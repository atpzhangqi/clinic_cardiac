# Daily Notes

这里放每日学习记录、查房后笔记、病例线索和当天想到的问题。

## 建议格式

- 按 `年份 / 年-月 / 年-月-日.md` 存放
- 每天只维护一个主记录页
- 当天不追求整理完，重点是先留下上下文

## 建议内容

- 今日学习重点
- 今日病例 / 场景
- 新学到的知识点
- 需要复习的问题
- 需要转入专题笔记的内容

## 创建方式

运行：

```bash
./scripts/new_daily_note.sh
```

也可以指定日期：

```bash
./scripts/new_daily_note.sh 2026-04-05
```
